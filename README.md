# DataSense — Automated Data Wrangling & Insight Generator

DataSense is a **CLI-based data analytics pipeline** that loads messy datasets, validates data quality, performs advanced wrangling, generates EDA visualizations, and exports insights automatically.

The project simulates a **real-world data analytics workflow** where raw datasets must be validated, cleaned, merged, analyzed, and visualized before generating reports.

---

# Project Goals

This project demonstrates an **end-to-end data analysis pipeline** including:

* Data validation
* Data cleaning
* Dataset merging
* Exploratory Data Analysis (EDA)
* Automated visualization
* Insight generation
* Data quality scoring
* CLI-based pipeline execution
* Git-tracked development workflow

---

# Workflow Pipeline

The project follows a structured **data pipeline architecture**:


# Installation

Clone the repository

```
git clone https://github.com/yourusername/datasense-project
```

Navigate to the project folder

```
cd datasense-project
```

Create virtual environment

```
python -m venv venv
```

Activate environment

Windows:

```
venv\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

---

# Running the Pipeline

Basic run

```
python main.py
```

Run with all features

```
python main.py --plots --export --summary
```

CLI options

| Command   | Function                     |
| --------- | ---------------------------- |
| --plots   | generate visualization plots |
| --export  | export aggregated tables     |
| --summary | print analysis insights      |
